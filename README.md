# NVRCMC - React Activity

Projeto em React + Vite com consumo de API usando `fetch` e navegação com `react-router-dom`.

## Requisitos da atividade

- Conexão com API usando `fetch` ou `axios`: ✅
- Rotas (mínimo 3: principal + 2): ✅

## Rotas da aplicação

- `/` -> Página principal (Home)
- `/todos` -> Lista de tarefas
- `/sobre` -> Página sobre o projeto

## APIs utilizadas

Dados públicos do JSONPlaceholder:

- `https://jsonplaceholder.typicode.com/users`
- `https://jsonplaceholder.typicode.com/posts`
- `https://jsonplaceholder.typicode.com/todos`

## Tecnologias

- React
- Vite
- React Router DOM
- JavaScript (ES6+)
